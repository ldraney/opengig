Documentation

Everything you need to know to use opengig - the zero-fee freelance marketplace.

Installation

opengig requires Node.js 18+ and Claude Code.

Quick Install

The fastest way to get started is using npx (no installation required):

npx opengig auth    # One-time authentication
npx opengig         # Launch the marketplace

Global Install

For frequent use, install globally:

npm install -g opengig

opengig auth
opengig
Note opengig launches Claude Code with marketplace tools configured. Make sure you have Claude Code installed first.

Authentication

opengig uses LinkedIn OAuth for identity verification. This ensures all users are real people with professional profiles.

Sign In

npx opengig auth

This will:

  1. Open your browser to LinkedIn
  2. Request permission to verify your identity
  3. Save your session locally (in ~/.opengig/)

Check Status

Once in Claude Code, you can check your auth status:

You: Am I logged in?

Claude: Yes! You're authenticated as Jane Smith (jane@example.com).
Your LinkedIn profile is verified.

Sign Out

To sign out, delete your local session:

rm -rf ~/.opengig

First Steps

After authentication, run npx opengig to launch Claude Code with marketplace tools.

Try these commands:

# Check your profile
You: What's my profile look like?

# Search for opportunities
You: Find React jobs, remote, $80-100/hr

# Post your availability
You: I'm available for Python/Django work, $90/hr

Finding Work

Search for job opportunities using natural language:

You: I'm looking for React Native work, remote, around $85/hr

Claude: Found 4 matching jobs! Here are the top ones:

1. **Mobile Developer** - FinTech Startup
   - $80-100/hr, fully remote
   - Skills: React Native, TypeScript
   - Posted 2 days ago

2. **Senior RN Engineer** - HealthTech
   - $90-120/hr, remote (US timezone)
   - Skills: React Native, Redux, GraphQL

Want me to help you reach out to any of these?

Search Tips

  • Be specific about skills: "React" vs "React Native" vs "React + Node"
  • Mention your rate range to filter better matches
  • Specify remote preference: "remote only", "hybrid OK", "on-site in NYC"

Listing Expiration Listings automatically expire after 30 days. You can renew them anytime by saying "renew my listing" or checking your listings with "show my listings".

MCP Tools Reference

opengig provides these tools to Claude Code:

Tool Description
auth_status Check login state and view your profile
create_listing Post a job or availability listing (expires in 30 days)
search_listings Search for jobs or talent with advanced filters
get_conversations List all your message threads
send_message Send a message to another user
get_messages Read messages in a conversation
share_contact Share your contact info with someone
get_my_listings View your listings with expiration status
renew_listing Renew a listing for 30 more days
deactivate_listing Remove one of your listings
save_search Save search criteria for alerts
get_saved_searches View your saved searches
run_saved_search Run a saved search and get matches
delete_saved_search Delete a saved search
get_notifications View recent notifications

Trust Model

opengig uses LinkedIn for identity verification:

  • LinkedIn OAuth - All users sign in with LinkedIn
  • Email Verification - Only accounts with verified emails
  • Real Profiles - Linked to actual LinkedIn identities
  • Consent-Based Sharing - Contact info only shared when you explicitly choose
Stay Safe While we verify LinkedIn identities, always do your own due diligence before sharing sensitive information or starting paid work. Use contracts and secure payment methods.

Troubleshooting

Authentication Issues

Problem: "Not authenticated" error

# Clear your session and re-authenticate
rm -rf ~/.opengig
npx opengig auth

Claude Code Not Found

Problem: opengig can't launch Claude Code

Make sure Claude Code is installed and in your PATH:

# Check if claude is available
which claude

# If not, install Claude Code from https://claude.ai/code

MCP Tools Not Loading

Problem: Claude doesn't have marketplace tools

Try restarting with a fresh session:

# Exit Claude Code and re-run
npx opengig
